데이터 분석 및 머신 러닝 모델 구축


 Development Content 

■ 팀 구성 및 역할
- 본인: 데이터 전처리, 학습 모델 검증, 데이터 시각화, 참고 자료 수집
- 팀원(1명): 주제 선정, 목적 정의, 학습 모델 선정, 보고서 작성

■ 프로젝트 설명
- 데이터 세트를 데이터 프레임으로 불러와서 결측치 제거 및 필요한 변수 선별을 통한 데이터 전처리
- 학습 모델의 성능 향상을 위한 데이터 스케일링을 통한 표준 정규화
- 변수 간 상관관계를 히트맵 등을 사용하여 시각화하여 분석
- 모델 구축 및 데이터 분류 후 학습
- 학습 모델별 혼동행렬을 통한 성능 평가 지표 분석
- 모델 성능 비교 및 결론 도출

■ 머신 러닝 학습 모델
Logistic Regression, Decision Tree, Random Forest

■ 성과
- 데이터 전처리, 분석, 머신 러닝 모델 구축 경험 획득
- 팀원 간의 협력과 커뮤니케이션 능력 향상
- 충분한 양과 품질이 좋은 데이터를 사용하면 머신 러닝 모델의 성능 향상이 가능하여 정확성과 예측 능력이 개선될 것으로 기대됨
