DASH BOARD PROJECT


 Development Content 

■ 팀 구성 및 역할
- 본인: PHP와 DB 연동, 출력 화면 구현, 시각화 차트 구현, 테스트용 샘플 데이터 생성
- 팀원(1명): 일정 관리 및 계획 수립, 보고서 작성, DB 설계 및 트리거 구현, 출력 화면 설계, 드롭 다운 버튼 구현

■ 프로젝트 (화면) 설명
- 생산 입력 패널: 모델 및 5가지 품질 항목 결과값 입력 시 데이터를 서버(로컬)에 저장
- 제품 데이터 리스트: 조회 조건(기간, 모델)에 따른 입력 데이터 레코드 출력
- 일간 생산 리포트: 조회 조건(기간, 모델)에 따른 일간 집계 데이터 출력
- 월간 생산 리포트: 조회 조건(기간, 모델)에 따른 월간 집계 데이터 출력, 모델별 월간 불량 누계 및 불량 항목 비중 차트 출력
- 생산 현황 대시보드: 월간 생산량, 불량 수, 불량률, 양품 수, 수율 테이블 및 차트 출력

■ 성과
- 웹사이트 및 DB 개발 구현 경험
- 팀원 간 협력과 커뮤니케이션 능력 향상
- 생산 현황과 불량을 즉시 파악하여 관리한다면 수율 증가 및 불량 감소 기대
